# Compile Proton

Download a precompiled build [here](https://github.com/ayvacs/Proton/releases) or build it yourself.

## Use a precompiled build

Download a precompiled build [here](https://github.com/ayvacs/Proton/releases). You only need to download the `.rbxm` file; import this to your Roblox game.

A script named `Proton` will be imported. You can parent it anywhere, but we suggest `ReplicatedStorage`.
