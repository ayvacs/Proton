# Installation

Installing Proton is as easy as 1, 2, 3!

1. First, download the [latest release](https://github.com/ayvacs/Proton/releases) of Proton from [its GitHub page](https://github.com/ayvacs/Proton). Under the `Assets` menu, download the file named `Proton.rbxm`.
2. Open Roblox Studio, and drag the downloaded `.rbxm` file into the Studio window. A `ModuleScript` named `Proton` will be added to your game; ensure it is parented to `ReplicatedStorage`.
3. Ensure that there are no other instances parented to `ReplicatedStorage` with the name `Proton`.
