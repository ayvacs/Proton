# All about Proton Properties

Proton Properties are entries that you may add to the properties table - the third argument of `new` - to enhance your code.

## Adding a `Child`, or `Children`

As you might've noticed, in the previous tutorial we used the `Children` property to create further Instances and parent them to our ScreenGui. That's one of many Proton Properties that you can use in your code.

Another one is `Child`, which functions similarily to `Children` - but it will parent a single Instance instead of an array of them.
